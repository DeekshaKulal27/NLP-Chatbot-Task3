#  NLP Chatbot using Hugging Face Transformers

##  Project Overview

This project implements a conversational chatbot using a pre-trained transformer model from Hugging Face. The chatbot can interact with users in natural language and generate responses dynamically.

---

##  Objective

To build an interactive chatbot using transformer-based NLP models that simulates human-like conversation.

---

##  Technologies Used

* Python
* Hugging Face Transformers
* PyTorch
* Jupyter Notebook / Google Colab

---

##  Model Used

* DialoGPT (microsoft/DialoGPT-medium)

---

##  Workflow

User Input → Tokenization → Model Processing → Response Generation → Output Display → Continuous Loop

---

##  Features

* Interactive chatbot system
* Dynamic response generation
* Maintains conversation context
* Handles multiple user inputs
* Exit functionality (`exit` / `quit`)

---

##  Sample Output

Chatbot: Hello! I am your AI assistant. How can I help you today?

You: Hello
Chatbot: Hello! Nice to meet you. How can I assist you today?

You: What is Artificial Intelligence?
Chatbot: Artificial Intelligence refers to the simulation of human intelligence by machines.

You: Who created Python?
Chatbot: Python was created by Guido van Rossum in 1991.

You: exit
Chatbot: Chatbot ends the conversation.
